# Heart Disease Analysis Dashboard Using Power BI

## Project Overview

This project analyzes heart disease patient data using Python and Power BI. The goal is to identify heart disease risk patterns, patient demographics, and key health indicators through data analysis and interactive dashboard visualizations.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Power BI
* Jupyter Notebook

## Dataset

The dataset contains 918 patient records with features such as age, gender, chest pain type, cholesterol, maximum heart rate, and heart disease target value.

## Dashboard Features

* Total patient count
* Heart disease distribution
* Gender analysis
* Chest pain type analysis
* Interactive filters

## Files Included

* Heart_Disease_Project.ipynb
* Heart_Disease_Dashboard.pbix
* cleaned_heart_data.csv
* dashboard_screenshot.png

## Project Purpose

This project demonstrates skills in data analysis, healthcare analytics, Power BI dashboarding, Python-based exploratory data analysis, and business intelligence reporting.
